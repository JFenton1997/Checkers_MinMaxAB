REQUIREMENTS
-----------------
- installation of pygame => https://www.pygame.org/wiki/GettingStarted
- Python 3.8.9 (version of python used to create the game, likely to run on majority of earlier 3.8 versions)

HOW TO RUN
-----------------
Run the main.py file in the source code, if pygame is installed correctly, everything will work
as intended.

(method i used to run was, using microsoft python extension for Visual Studio Code, and running the main)
